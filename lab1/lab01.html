Ball Python

งูบอลไพธ่อนมีชื่อวิทยาศาสตร์ว่า Python regius มีถิ่นกำเนิดจากทวีปแอฟริกา โดยพบตั้งแต่แอฟริกากลางไปจนถึงแอฟริกาตะวันตกโดยปกติแล้วงูบอลไพธ่อนมักจะใช้ ชีวิตอยู่บนพื้นดินมากกว่าที่จะปีนขึ้นไปอยู่บนต้นไม้ แต่ก็ไม่ใช่ว่าจะปีนต้นไม้ไม่เป็นบ่อย ครั้งที่พบงูบอลไพธ่อนอยู่บนต้นไม้ก็มี งูบอลไพธ่อนนั้นได้ถูกนำมาเพาะพันธุ์เพื่อเป็นสัตว์ เลี้ยงมานานพอสมควรแล้วเนื่องจากนิสัยที่ไม่ก้าวร้าว นอกจากนี้ยังมีสีสันลวดลายสวยงาม อีกทั้งยังมีขนาดไม่ใหญ่เมื่อเทียบกับงูในกลุ่ม Python ชนิดอื่นๆ ในยุโรปมักจะเรียกงูชนิด นี้ว่า Royal Python ในขณะที่ทางฝั่งอเมริกาจะเรียกว่า Ball Python ซึ่งในเมืองไทย เราก็เรียกชื่อเดียวกับอเมริกา
